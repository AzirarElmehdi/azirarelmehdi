## Projets

### 🛒 AZMethods Shop
E-commerce fullstack avec moteur de promotions, panel admin protégé et architecture Context-driven.

- Moteur de prix cumulatif (item → catégorie → global) isolé en pure function
- Pagination côté serveur via Supabase `.range()`
- Contexts globaux (`ShopContext`, `CartContext`) pour zéro re-fetch inutile
- CI/CD via GitHub Actions + déploiement Vercel

**Stack :** React 19 · React Router v7 · Supabase · Tailwind CSS

[→ Voir le site](https://le-shop.vercel.app) | [→ Repo](https://github.com/AzirarElmehdi/LeShop)

### 🎮 Unity Grid System — A* Pathfinding
Implémentation manuelle de l'algorithme A* sur une grille générée procéduralement.

- Génération de grille avec détection d'obstacles par `Physics.CheckSphere`
- Algorithme A* from scratch avec heuristique Octile Distance
- Agent autonome qui calcule et suit le chemin optimal en évitant les obstacles
- Architecture Single Responsibility : Node / GridManager / Pathfinding / PNJController

**Stack :** Unity 6 (LTS) · C#

[→ Repo](https://github.com/AzirarElmehdi/Unity-Grid-System)
